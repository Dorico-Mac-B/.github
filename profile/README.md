## Overview

Dorico Advanced is a professional music notation application from Steinberg that creates beautifully engraved scores and parts with an intelligent layout engine that handles the complex spacing, alignment, and positioning decisions that professional music engraving requires. It was developed by the team that previously created Sibelius, bringing decades of notation software experience to a modern architecture designed without the legacy constraints of older applications. The application combines notation creation, playback, and engraving in a unified environment that serves composers, arrangers, educators, and music publishers.

The engraving engine automatically handles note spacing, staff spacing, system breaks, page turns, and collision avoidance with typographic intelligence that produces publication-quality output without manual adjustment in most cases. Default engraving rules follow established conventions from authoritative music typography references, and every rule is configurable for house styles that deviate from convention. The automatic layout produces results that rival hand-engraved scores, reducing the time between composition and print-ready output.

Input methods support MIDI keyboard entry, computer keyboard step input, mouse input, and MusicXML import, accommodating every composer's preferred notation entry workflow. Note input is pitch-before-duration or duration-before-pitch depending on user preference, and the input system intelligently handles voice assignment, tie chaining, and tuplet creation. Repitch mode allows changing pitches of existing rhythms without re-entering durations.

Playback with expression maps and NotePerformer integration renders notation with musical intelligence that handles dynamics, articulations, tempo variations, and phrasing rather than producing the mechanical playback that notation software traditionally delivers. The playback engine interprets notation symbols — hairpins, staccato, legato, trills — and translates them into realistic performance data. HALion Sonic SE provides a capable general MIDI sound set, while VST instrument support enables playback through any virtual instrument library.

For composers, arrangers, music educators, and publishers who need notation software that produces publication-quality engraving with intelligent automation, Dorico Advanced is the modern notation application that combines beautiful output with a workflow designed for how musicians actually create scores. Its fresh architecture enables features and workflows that legacy notation applications cannot implement without fundamental restructuring.

## Performance

Dorico balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
